## Project : MCyouyou World简介
Project: MCyouyou World是由MCyouyou公益服务器发起的一项开放MC项目。  
该项目是一个综合性MC服务建设项目，无偿对国内公益服务器提供接入使用。

## 项目内容
该项目为国内公益MC服务器提供免费推广，技术支持，静态文件托管，服务器托管等。

有关该项目的详情，请在[MCyouyou Wiki](https://wiki.mcyou.cn)内查看
